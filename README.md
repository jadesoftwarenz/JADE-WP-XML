# Example files for the XML in JADE White Paper
This repository the files referenced in the XML in JADE White Paper. It showcases the XML framework in JADE, which has been implemented to enable rapid development of XML applications and XML components.

## Documentation Guide
The XML in JADE White Paper that references this repository may be found here: <a href="https://www.jadeworld.com/pdf/white-papers/2018/WP_XML.pdf">XML in JADE White Paper</a>

## Getting Started
It is recommended to start by reading the <a href="https://www.jadeworld.com/pdf/white-papers/2018/WP_XML.pdf">XML in JADE White Paper</a>, which contains detailed instructions on using the demonstration schema.

### Prerequisites
Before you can load the schemas, you will need JADE 2018 installed.
Instructions for installing JADE may be found at <a href="https://www.jadeworld.com/downloads/tech/manuals/2018/InstallConfig.pdf">JADE install instructions.</a>

### Loading the schemas by downloading a .zip file.
Instructions may be found in the <a href="https://www.jadeworld.com/pdf/white-papers/2018/WP_XML.pdf">XML in JADE White Paper</a> which references this repository.

### Loading the schemas using JADE Git Integration
Instructions for using JADE Git Integration may be found here: <a href="">Jade Git Integration guide.</a>

## Frequently Asked Questions
**Q.** Can I contribute to or change these schemas?
> The schemas contained in this repository are for demonstration purposes and as such are not open to pull requests. However, you are welcome to create a fork and make changes to your own copy, subject to our license ([LICENSE.txt](LICENSE.txt))

**Q.** What is Git and how do I use it?
> Git is a version control system that is external to JADE and as such the JADE product information library does not cover the Git technology itself. For information about JADE's implementation of Git source control, see <a href="">JADE Source Control.</a> For an example resource on the Git technology, we recommend the Pro Git book. The Pro Git book is written by Scott Chacon and Ben Straub, who are not affiliated with Jade in any way. An online version of the book may be found at <a href="https://git-scm.com/book/en/v2">Pro Git</a>
